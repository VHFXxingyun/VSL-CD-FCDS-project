V0.1 For experimental verification of the feasibility of this project.
The first challenge lies in achieving communication between the Raspberry Pi 5 and the Waveshare ESP32 expansion board. 
Due to insufficient understanding of serial communication and encountered issues, communication was initially implemented via GPIO-level signals. 
This version uses four signal channels to control the two steering motors in four directions, preliminarily validating the feasibility of the project. 
However, this is only an early debugging version, and other modules have not yet been integrated.
